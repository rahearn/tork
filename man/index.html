tork(1)
Continuous testing tool for Ruby
tork-master(1)
absorbs overhead and runs tests
tork-notify(1)
notifies you of test status changes
tork-herald(1)
reports modified files
tork-remote(1)
controls tork(1) programs
tork-engine(1)
wraps tork-master(1) with bookkeeping
tork-driver(1)
drives tork-engine(1) when files change
